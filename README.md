This repository contains the source code for Flash. Flash is a geoguessr-like game based in Singapore, where the player must guess buildings in Singapore based on their 3d shape in google maps.

To install the app, download the apk file in this repository, or from this google drive https://drive.google.com/drive/folders/1V6SxXgICvWzszbtO5rJ9DfhqdeIqmnpl?usp=sharing
